# Note API

Provides functions for managing notes.

## Files

- `index.js`

## Functions

### `getNoteList(params)`
Retrieves a list of notes.

### `getNoteDetail(params)`
Retrieves details of a specific note.

### `insertNote(data)`
Creates a new note.

### `updateNote(data)`
Updates an existing note.

### `deleteNote(data)`
Deletes a note.

### `updateNoteReadYn(data)`
Updates the read status of a note.
