# Coop API

Provides functions for managing cooperative partners.

## Files

- `index.js`

## Functions

### `fetchCoopList(params)`
Retrieves a list of cooperative partners.

### `fetchCoopDetail(params)`
Retrieves details of a specific cooperative partner.

### `insertCoop(data)`
Creates a new cooperative partner.

### `updateCoop(data)`
Updates an existing cooperative partner.

### `deleteCoop(data)`
Deletes a cooperative partner.

### `fetchCoopIpList(params)`
Retrieves a list of IPs for a cooperative partner.

### `insertCoopIp(data)`
Adds an IP for a cooperative partner.

### `deleteCoopIp(data)`
Deletes an IP for a cooperative partner.

### `fetchCoopBoardList(params)`
Retrieves a list of board posts for a cooperative partner.

### `fetchCoopBoardDetail(params)`
Retrieves details of a board post for a cooperative partner.

### `insertCoopBoard(data)`
Creates a new board post for a cooperative partner.

### `updateCoopBoard(data)`
Updates an existing board post for a cooperative partner.

### `deleteCoopBoard(data)`
Deletes a board post for a cooperative partner.

### `fetchCoopOrderList(params)`
Retrieves a list of orders for a cooperative partner.

### `fetchCoopPayDetailList(params)`
Retrieves payment details for a cooperative partner.
