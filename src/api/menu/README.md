# Menu API

Provides functions for managing the menu structure.

## Files

- `index.js`

## Functions

### `fetchMenuTree()`
Retrieves the menu tree structure.

### `fetchPassMenuTree()`
Retrieves the pass menu tree structure.

### `fetchDetailMenu(menuId)`
Retrieves details of a specific menu item.

### `fetchPassDetailMenu(menuId)`
Retrieves details of a specific pass menu item.

### `updateMenu(menuData)`
Updates a menu item.

### `updatePassMenu(menuData)`
Updates a pass menu item.

### `deleteMenu(menuId)`
Deletes a menu item.

### `deletePassMenu(menuId)`
Deletes a pass menu item.

### `checkMenuId(menuId)`
Checks if a menu ID is valid/available.

### `getMaxMenuId()`
Retrieves the maximum menu ID.

### `getPassMaxMenuId()`
Retrieves the maximum pass menu ID.

### `insertMenu(menuData)`
Creates a new menu item.

### `insertPassMenu(menuData)`
Creates a new pass menu item.
